2. The ADT node list

Complete the implementation of the class NodePositionList given in §6.2.4, pp. 252–257, of our main textbook. This class uses a doubly-linked list for its underlying construct. The class DNode.java defines a node of such a doubly-linked list and implements the interface Position. The class NodePositionList implements the ADT node list whose operations are defined in the interface PositionList. This class consists of the the following thirteen methods.
  1. size()—returns the number of elements of the list.
  2. isEmpty()—determines if the list is empty.
  3. first()—returns the first position.
  4. last()—returns the last position.
  5. prev(p)—returns the position that precedes a position p.
  6. next(p)—returns the position that follows a position p.
  7. set(p, e)—replaces the element at a position p with an element e and returns the element replaced.
  8. addFirst(e)—inserts an element e as the first element.
  9. addLast(e)—inserts an element e as the last element.
  10. addBefore(p, e)—inserts an element e before a position p.
  11. addAfter(p, e)—inserts an element e after a position p.
  12. remove(p)—removes the element at a position p and returns the element removed.
  13. iterator()—returns an iterator of the elements.
All but the methods last(), next(), addLast() and addAfter() are already implemented. You are to implement the remaining four methods. Be sure to throw appropriate exceptions. Then add a test driver program to the file Tester.java to test your NodePositionList class. Your driver program should at least perform the tests shown in Example 6.3, p. 250, of our main textbook. The class Tester has a static method named print() that uses an iterator to print the elements of a given list. Your driver program should invoke this method to print the content of the list after every list operation.

3. Iterators

The class NodePositionList has a method named iterator() that returns an iterator of all the elements defined in the class ElementIterator. Now, implement two additional iterators as follows.

The class ReverseElementIterator. In a new class named ReverseElementIterator, implement an iterator of all the elements in the reverse order. This class should implement the interface Iterator. Then, in the class NodePositionList, add a method named reverseIterator() that returns an iterator object of the class ReverseElementIterator. Add code to Tester.java to test your implementation.

The class EveryOtherElementIterator. In a new class named EveryOtherElementIterator, implement an iterator of every other element (starting from the first element). This class should also implement the interface Iterator. Then, in the class NodePositionList, add a method named everyOtherIterator() that returns an iterator object of the class EveryOtherElementIterator. Add code to Tester.java to test your implementation. Once completed, save a snapshot of your test run in a text file named lab6.out.
