2. Braces matching

In every Java expression, each opening brace must be matched with a closing brace of the same kind. In particular, every opening “(”, “{”, “[” must be closed by its matching “)”, “}”, “]”, respectively, before a previously opened “(”, “{” or “[” is closed. You are to write a command-line program that will read a Java source file and checks if all the opening and closing braces properly match.

The class CheckBraces. Your Java application should be defined in a class named CheckBraces and must run via the following command: 

java CheckBraces filename
Here, the argument filename specifies the name of a Java source file to be processed. The class CheckBraces is partially completed. In particular, this program already has a code segment to read a text file given in the command line into a string named inString. It also has a code segment to echo print inString. You are to analyze the text within the string inString. Your program should terminate upon finding an error, either an unmatched or missing brace, and indicate where the error is by a character number (for example, character 7 means the seventh character from the beginning).

Testing correctness. To check your correctness, run your program on the files

ValidClass.java
UnmatchedBrace.java
MissingBrace.java
(which may be downloaded by clicking these links from this Laboratory 5 webpage). Save a snapshot of each test run in a text file named lab5.out. For UnmatchedBrace.java, it should behave in the following way: 
miyazaki@lab:~/cpsc215/lab5$ java CheckBraces UnmatchedBrace.java 
/**
 * This class has an unmatched brace.
 */

public class UnmatchedBrace {
  int arr[] = new int[25];
  public UnmatchedBrace() {
    init();
  }
  public void init() {
    for (int k = 0; k < arr.length; k++)
      arr[k} = 10;
  }
}

Unmatched brace at character 223: Found } expecting ].
For MissingBrace.java, it should behave in the following way: 
miyazaki@lab:~/cpsc215/lab5$ java CheckBraces MissingBrace.java 
/**
 * This class has a missing brace.
 */

public class MissingBrace }
  int arr[] = new int[25];
  public MissingBrace() {
    init();
  }
  public void init() {
    for (int k = 0; k < arr.length; k++)
      arr[k] = 10;
  }
}
